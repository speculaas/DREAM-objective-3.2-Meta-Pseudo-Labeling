# Synthetic AM Object Inventory

This folder contains the generated GitHub Pages artifact for the dataset object inventory.

Generated files:

- `index.html`: searchable inventory webpage
- `object_inventory.csv`: tabular inventory export
- `object_inventory.json`: structured inventory export
- `.nojekyll`: disables Jekyll processing on GitHub Pages

To publish this page:

1. Commit the `docs/` folder.
2. Go to GitHub repository settings.
3. Open **Pages**.
4. Select source branch, usually `main`.
5. Select `/docs` as the publishing folder.
